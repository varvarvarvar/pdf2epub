# PDF 2 TXT 2 EPUB

Sometimes ebooks cannot display .pdf properly and that's when pdf2txt2epub comes in handy.
<br>
<br>
Convert .pdf documents to .txt and .epub using Optical Character Recognition in 4 simple steps:

1. Convert input `.pdf` to PIL Images.
2. Recognize and postprocess text on PIL Images using `Tesseract OCR`.
3. Save the result text to `.txt`.
4. Convert `.txt` to `.epub` using `Pandoc`.
